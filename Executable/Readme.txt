To convert
1) Copy your CSV in same folder
2) Open meta.properties
3) Change csv to your csv name 
4) Open cmd in same location
5) run the following command 
*********************************************
Command to convert using single thread :  java -jar -Xms4096M -Xmx6144M csv2html.jar 1  
*********************************************
 
*********************************************
Command to convert using multi thread :  java -jar -Xms4096M -Xmx6144M csv2html.jar 2 
*********************************************
